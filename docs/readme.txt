CherryTree is available with the base repositories of Ubuntu. You can install it using the following command :

  #sudo apt-get install cherrytree


Note that the CherryTree Text Editor available with the Ubuntu base repositories are outdated. It's recommended to install by adding PPA :

  #sudo add-apt-repository ppa:giuspen/ppa
  #sudo apt-get update
  #sudo apt-get install cherrytree
